Skills

Html

adalah bahasa markup yang digunakan untuk membuat dan menyusun struktur halaman web. HTML menggunakan elemen-elemen yang disebut tag untuk menandai berbagai bagian konten, seperti teks, gambar, tautan, dan tabel.

Css

adalah bahasa yang digunakan untuk mendesain dan mengatur tampilan halaman web, seperti warna, ukuran, tata letak, dan animasi. CSS memisahkan struktur HTML dari desain, membuat pengembangan web lebih terorganisir dan efisien.

Java Script

adalah bahasa pemrograman yang digunakan untuk membuat halaman web interaktif, seperti animasi, formulir dinamis, dan perubahan konten tanpa memuat ulang halaman.
